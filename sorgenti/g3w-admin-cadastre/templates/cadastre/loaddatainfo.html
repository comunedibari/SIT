Dati DOCFA

Per caricare i dati DOCFA deve essere creato un file .zip al cui interno siano contenuti i seguenti file .zip scaricati dal portale AGEA

Solo un tipo di file deve essere modificato:

  • *_DOC_*.zip: possono esser anche più di uno e al suo interno deve contenere i file .DAT da cui, mediante apposito software fornito da AGEA, è ricavare il corrispondente file .pdf che deve essere anch'esso caricato all'interno del file .zip (di seguito un esempio di file, B455_201609_DOC_1.zip).

  • *_DC_*.zip: cosi come viene scaricato/i dal portale.
  • *_DM_PL_*.zip: contenente i file delle planimetrie cosi come viene scaricato/i dal portale.
